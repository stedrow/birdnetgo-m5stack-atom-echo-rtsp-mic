Atom Echo — RTSP Mic for BirdNET-Go

Flash your M5Stack Atom Echo directly from Chrome.
No drivers or tools required.

WebSerial is only supported in Chrome or Edge on desktop. Please open this page in Chrome to flash your device.
  1. Connect your Atom Echo via USB
  2. Select a firmware release below
  3. Click Install and select the serial port
  4. Wait for flashing to complete (~30 seconds)
  5. Connect to ESP32-RTSP-Mic-AP to configure WiFi
  6. Stream at rtsp://atomecho.local:8554/audio
